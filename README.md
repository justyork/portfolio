# Iurii Shpynev — Portfolio

Bilingual static portfolio / expanded CV for `justyork.dev`.

## Stack

- Static-first HTML/CSS generated by Node.js scripts
- No runtime dependencies
- Fully server-rendered/static markup, indexable without JavaScript
- English + Russian routes with canonical/hreflang tags
- `sitemap.xml` and `robots.txt` generated during build
- Sanitized PDF CV generated during build

## Commands

```bash
npm run build
npm run check
npm run dev
```

Build output is written to `dist/`. The site is SEO-indexable by default: pages are static HTML, include `index, follow`, canonical and hreflang tags, and expose `sitemap.xml` from `robots.txt`.

## Routes

- `/` redirects to `/en/`
- `/en/` — English portfolio
- `/ru/` — Russian portfolio
- `/cv/iurii-shpynev-cv.html` — printable CV
- `/cv/iurii-shpynev-cv.pdf` — sanitized downloadable CV
- `/sitemap.xml` — search engine sitemap
- `/robots.txt` — crawler policy

## Public contact policy

Show LinkedIn, Telegram and email. Do **not** publish phone number.
